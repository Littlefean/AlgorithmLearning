## 快速生成数据集

只要写一个图片，就能生成对应一百个小图片，方便快速生成大量数据集。如何做到？

可以进行变换的方法：

1. 大小整体缩放
2. 上下左右略微平移
3. 左右略微旋转
4. 横向左右梯形拉扯

可以只通过一个图片，生成将近一千个类似的图片。
